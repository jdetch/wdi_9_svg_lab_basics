SVG + CSS3

We can use CSS3 to style our SVG files. This helps us get all those terrible styling attributes out of our markup and into a stylesheet where they belong. However, you should still keep most of your positioning and sizing attributes in the markup directly where you can easily manipulate them with JavaScript (hint hint, d3).

Note the style.css file. See how this rectangle is still green with a stroke on it?

We can select using elements, ids and classes just like normal with CSS.
