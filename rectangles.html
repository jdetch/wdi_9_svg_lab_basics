Boxes, Squares and Rectangles

Rectangles are drawn with the 'rect' element tag.

Attributes include:

  • width
  • height
  • x/y - These set the left/top position of the rectangle
  • style:
    • ry/ry - sets the corner radiuses
    • fill - fill color
    • stroke - border stroke color
    • stroke-width - stroke width
    • opacity - amount of transparency. Between 0 and 1.
